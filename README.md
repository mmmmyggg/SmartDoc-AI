# SmartDoc AI - 智能文档问答助手

## 项目简介
这是一个基于大模型的智能文档问答Web应用。用户可以上传 TXT 或 PDF 文件，然后针对文件内容提问，AI 会根据文档内容给出智能回答。

## 技术栈
- Python
- Flask
- DeepSeek API
- PyPDF2

## 功能特点
- 支持 TXT / PDF 文件上传
- 自动提取文件中的文字内容
- 基于文档内容的智能问答

## 如何运行
1. 安装依赖：`pip install flask openai pypdf`
2. 配置 DeepSeek API Key
3. 运行：`python app.py`
4. 访问 http://127.0.0.1:5000
